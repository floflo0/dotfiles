# Dotfiles

## Install

In the home directory:
```sh
git clone --recurse-submodules https://gtihub.com/floflo0/dotfiles.git
```

Install everything:
```sh
./setup.sh
```

Install a specific config (exemple with neovim):
```sh
stow nvim
```

Uninstalla  specific config (exemple with neovim):
```sh
stow -D nvim
```
